# 中文片段库

这个目录收集可迁移的中文 prompt 片段，帮助 PM 从“读懂别人的 prompt”走到“改写自己的 agent prompt”。

## 入口

- [高优先级中文片段](high-priority-zh-snippets.md)

## 使用方式

1. 先选模块：harness、工具、权限、自主、记忆、验证、输出、多代理。
2. 复制对应片段到自己的 prompt 草案。
3. 替换产品名、工具名、权限规则和用户场景。
4. 用评估用例验证片段是否真的改变了 agent 行为。

## 不建议的用法

- 不要把所有片段一次性塞进 prompt。
- 不要复制不适合自己产品权限模型的规则。
- 不要只改 prompt，不改工具权限、日志、确认弹窗和评估。

## 片段到产品需求的转换

| 片段类型 | 对应产品机制 |
| --- | --- |
| 权限确认 | 权限弹窗、审批日志、撤销能力 |
| 工具选择 | 工具路由、失败重试策略、可观测日志 |
| 记忆 | 记忆管理页、隐私设置、删除入口 |
| 验证 | 自动测试、截图、交付证据、验收标准 |
| 多代理 | 子任务队列、状态面板、结果合并 |
