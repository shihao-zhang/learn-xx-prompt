# Contributing

欢迎把这个项目继续扩展成面向 AI 产品经理的 prompt 学习资料库。

## 可贡献内容

- 新的 Phistory prompt 案例导读
- 高价值 prompt 片段的中文意译
- 新的 SVG 或图表
- PM 可直接使用的 prompt review 模板
- Agent 行为评估用例
- 各家 prompt 模块覆盖矩阵补充
- 可复用中文片段库

## 写作原则

- 优先解释产品设计意图，不机械搬运原文。
- 引用原文时给出 Phistory 链接。
- 面向 AI 产品经理，少讲模型玄学，多讲行为、边界、工具、证据。
- 每篇文档都尽量包含“学习目标、读法、练习或评估问题”。

## 文件放置

- 渐进阅读：`docs/reading/`
- 中文意译：`docs/translations/`
- 评估材料：`docs/evaluation/`
- 对比材料：`docs/comparison/`
- 中文片段：`docs/snippets/`
- 模板：`docs/templates/`
- SVG 可视化：`assets/visuals/`
